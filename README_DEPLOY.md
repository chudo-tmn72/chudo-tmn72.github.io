# ЧУДО — готовая сборка сайта v3

Финальная сборка после ревью руководителя разработки и последнего патча.

## Что внутри
- `index.html` — одностраничный лендинг.
- `styles.css` — вся стилизация и адаптив.
- `script.js` — мобильное меню.
- `assets/` — ассеты из переданного архива, переложены из `site-assets/` в `assets/`.
- `assets/previews/og-image.jpg` — отдельная Open Graph картинка 1200×630 для предпросмотра в соцсетях.
- `assets/favicon.png` и `assets/apple-touch-icon.png` — иконки сайта.

## Что изменено в v3
- Header уходит в бургер-меню уже на ширине до `1080px`, чтобы не сжиматься на планшетах и небольших ноутбуках.
- Убрано визуальное дублирование кнопки `VK Магазин VK`: теперь используется аккуратная формулировка `VK Магазин` / `Магазин`.
- Добавлены `aria-label` на крупные карточки-ссылки: «Девочкам», «Мальчикам», «По росту» и три ростовки.
- Добавлены SEO/соцмета:
  - `canonical`;
  - `og:url`;
  - абсолютный `og:image`.
- JS мобильного меню синхронизирован с новым breakpoint `1080px`.

## Как залить на GitHub Pages
1. Откройте репозиторий сайта `chudo-tmn72.github.io`.
2. Сохраните текущие файлы на всякий случай.
3. Замените в корне репозитория файлы:
   - `index.html`
   - `styles.css`
   - `script.js`
4. Добавьте/замените папку `assets/` целиком в корне репозитория.
5. Сделайте commit/push.
6. После публикации проверьте сайт на телефоне и компьютере.

## Что проверено в сборке
- Все основные смысловые тексты вынесены в HTML.
- Hero использует `hero-desktop.webp` и `hero-mobile.webp`.
- На странице есть 3 карточки: «Девочкам», «Мальчикам», «По росту».
- Раздел «Образы» не выводится.
- Блок «По росту» содержит 3 ростовки и VK-ссылки.
- Иконки преимуществ и шагов заказа подключены отдельными файлами.
- Крупные изображения ниже первого экрана используют `loading="lazy"`.
- Подключены meta title, description, canonical и Open Graph.
- На мобильном/планшете в бургер-меню есть отдельная CTA-ссылка «Магазин».
- Закрытое мобильное меню скрывается визуально и через `aria-hidden`.
- Ссылка «Написать в VK» ведёт на группу `https://vk.com/chudo_tmn72` как более стабильный вариант.
