# QA checklist — ЧУДО site v3

## Автоматически проверено
- Локальные пути к `index.html`, `styles.css`, `script.js`, картинкам и иконкам.
- Наличие `favicon.png`, `apple-touch-icon.png`, `og-image.jpg`.
- Размер `og-image.jpg`: 1200×630.
- Все изображения открываются без ошибок.
- CSS: базовая проверка баланса фигурных скобок.
- JS: `node --check script.js` без синтаксических ошибок.
- Старый breakpoint `880px` удалён, новый breakpoint `1080px` есть в CSS и JS.
- Раздел/ассеты `Образы` не подключены в HTML.
- Обязательные VK-ссылки присутствуют.
- Внешние ссылки имеют `target="_blank"` и `rel="noopener"`.
- `canonical`, `og:url`, `og:image` присутствуют.
- На карточки категорий и ростовок добавлены `aria-label`.

## Проверено через headless-браузер на inline-сборке
- 390px: нет горизонтального скролла, меню скрыто до клика, `aria-hidden=true`.
- 768px: нет горизонтального скролла, меню скрыто до клика, `aria-hidden=true`.
- 1080px: включается бургер-меню, нет горизонтального скролла.
- 1366px: desktop-шапка, CTA в header виден, nav не скрыт.
- Открытие/закрытие меню по клику и Escape работает.

## Проверить вручную после публикации
- Реальные клики VK на телефоне: маркет, разделы по полу, ростовки, доставка, как заказать.
- Предпросмотр ссылки в VK/Telegram после обновления кэша.
- Первый экран на реальном iPhone/Android.
